School TimeTable
Generator

An automated, conflict-free timetable builder for schools and colleges — built with PHP, MySQL, HTML & CSS.

🐘 PHP 🗄️ MySQL 🌐 HTML 🎨 CSS
The Problem

Why does this project exist?

1

Making school timetables manually takes hours and is highly error-prone — teachers get double-booked, periods overlap, lunch breaks are missed.

2

When a school has multiple sections (10-A, 10-B, 10-C), each needs a separate timetable — multiplying the effort significantly.

3

Different sections sometimes need different teachers for the same subject — managing this manually leads to constant confusion.

4

There is no free, simple, web-based tool that solves all of this in one place without requiring any installation.

Teacher Override per Section

No Double Booking

Core Logic

The Scheduling Algorithm

// For each day → each class → each time slot:
foreach ($days as $day) {
  foreach ($classes as $class) {
    foreach ($slots as $slot) {
      // Find valid candidates (subject not used today, quota > 0,
      // teacher not on lunch, teacher not already booked this slot)
      $cands = findCandidates($subjects, $booked, $slot);
      // Pick subject with highest remaining quota (greedy)
      usort($cands, fn($a,$b) => $b['quota'] - $a['quota']);
      $quota[$class][$subject]--;  // decrement
      $booked[$day][$slot][$teacher] = true;
    }
  }
}

Conflict Prevention

$booked[day][slot][teacher] = true ensures no teacher is assigned twice in the same slot across different classes.

Graceful Fallback

If no valid candidate exists for a slot, it automatically becomes a Free Period instead of crashing.
